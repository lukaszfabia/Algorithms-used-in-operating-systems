### Simulation of access to disc. ###
Simulation includes 6 strategies: 
* First come first served `FCFS`
* Short seek time first `SSTF`
* Circular Scan `CSCAN`
* Scan `SCAN`
* Fd Scan `FDSCAN`
* Earliest deadline first `EDF`
